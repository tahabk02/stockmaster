# STOCKMASTER-PRO

Monorepo pour une plateforme de gestion de stock multi-tenant :

- `backend/` : API Node.js + Express + TypeScript
- `frontend-web/` : SPA React + Vite + TS
- `mobile-app/` : App mobile React Native + Expo
- `infrastructure/` : scripts et config de déploiement
- `docs/` : documentation API et déploiement

## Démarrage rapide

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend Web

```bash
cd frontend-web
npm install
npm run dev
```

### Mobile App

```bash
cd mobile-app
npm install
npm run start
```
